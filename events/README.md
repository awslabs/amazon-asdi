# Amazon Sustainability Data Initiative Events

----

At re:Invent,  the Amazon Sustainability Data Initiative and AWS Data Exchange, together with AWS Sustainability, is hosting **Code Green: Hacking on Amazon Sustainability Data Initiative datasets**, a sustainabillty hackathon and workshop for learning about and building on ASDI data. [Learn more here.](https://github.com/awslabs/amazon-asdi/tree/master/code-green)
